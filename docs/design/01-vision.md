# Vision & Core Pillars

## Elevator Pitch

A first-person party-based dungeon crawler set in a grim, low-magic world where factions scheme, the dead don't stay dead, and your choices reshape the balance of power. Low-poly 3D exploration, turn-based tactical combat, and a flexible roster system — built for the browser.

## Core Pillars

### 1. The world moves without you
Factions pursue their own goals on timers. Miss an opportunity and someone else seizes it. Quest states change, alliances shift, dungeons get claimed or emptied by rivals.

### 2. Death is a resource, not a boundary
Necromancy is infrastructure, not taboo. The world runs on it — labor, war, governance. The moral weight isn't "magic bad" but "what are you willing to spend." Characters can die permanently. Resurrection has costs beyond gold.

### 3. Composition over class
Characters start with a class archetype but diverge through level-up choices. The roster/bench system rewards experimentation. No single "correct" party — different dungeons and faction encounters reward different builds.

### 4. Earned discovery
No quest markers or hand-holding. The world communicates through environmental storytelling, NPC rumors, item lore, and faction behavior. Mapping matters. Paying attention pays off.

## Target Platform
- Web browser (HTML5/WebGL)
- Low-poly 3D first-person dungeon view
- Grid-based step/turn movement

## Campaign Structure
- Mid-sized campaigns: 8-15 hours per run
- Semi-randomized via six campaign rolls + LLM arrangement of authored content
- High replayability through modular faction roles and procedural narrative
