## Save version chain (consolidated)

| v | Adds | Owning spec |
|---|---|---|
| 1 | Phase 1 base | pre-existing |
| 2 | Backpack, LifeState, Cache, Storage, gold, tithe | inventory + combat-state |
| 3 | missions, worldFlags, worldTurn | quest-mission |
| 4 | journal entries, replayBufferIds | field-notes-journal |
| 5 | npcVars, npcDispositions, activeDialogue | dialogue-system |
| 6 | turnHistory, townVisitId, lastDowntimeVisitId | world-clock |
| 7 | rngTreeRoot, stateHash | determinism-replay |
| 8 | factions, repHistory, campaignRolls | faction-system |
| 9 | vendor runtime state | vendor-economy |
| 10 | crafting known recipes | crafting |
| 11 | rumors heard + verified | rumor-system |

Migration chain in `src/engine/RPC.Engine/Save/Migrations/`. Each step pure `JObject → JObject`. Save-format spec §4.

## Wire protocol versioning

Single `v: 1` envelope. Phase 2 adds state-diff via `state_update.full: false`. Phase 2+ adds new action/event types within v=1 (additive only). v=2 reserved for breaking.

## Phase build-out (concrete order)

### Phase 1 (current focus)

Foundation: design-system tokens + layered UI shells + primitives.
Domain: inventory-model, combat-state, ability-system, damage-status, combat-ai (3 archetypes), party-formation, encounter-generation, progression-system, vendor-economy, dungeon-assembly, quest-mission (3 starter missions).
Tech: websocket-protocol v1, save-format v1→v9 migrators, settings-keybinds (client-only), determinism-replay (RngTree + action log), content-pipeline (RPK + schemas), error-recovery (try/catch + autosave + corrupt fallback), telemetry (local only).
Tests: testing-strategy gates active.
Accessibility: a11y baseline (keyboard, ARIA, contrast, reduced-motion).

### Phase 1.5

audio-architecture (engine + ~30 cues), field-notes-journal (synergy engine + 5 synergies + replay), dialogue-system (linear shopkeep subset), onboarding (nudges + help + diegetic teach), lore-document (~18 docs), crafting (downtime + 8 recipes), inventory-model expansion (Bloom decay, equip as quick-action), combat-state-extension (Surgeon revive), combat-ai (quick actions, +2 archetypes), faction-system (Bureau + Convocation), world-clock (15-turn campaign live).

### Phase 2

Full faction-system (all 5), world-clock full reactivity, dialogue full grammar (skill checks, combat triggers, faction rep), rumor-system, state-diff-protocol, content-pipeline hot-reload + dev tooling, asset-pipeline real assets, dev-experience (overlay + scenarios + REPL), localization (second locale end-to-end), photino-lifecycle full, release-pipeline (signed installers, auto-update), error-recovery (supervisor wrapper, crash report), multi-slot saves, telemetry view/export UI.

### Phase 3

LLM campaign generation, all 8 dungeon templates, synergy library (40-50), combat-ai Unaccounted archetype, telemetry opt-in upload, modding system + workshop, audio per-template ambient, branch-specific SFX, Unaccounted audio breaks, tournament/share systems, new-game-plus.
